# TechNexis

Modern AI Tools + Tech Blog website built with Node.js + Express.

## Run locally
```bash
npm install
npm start
```

Open `http://localhost:3000`

## Deploy on Render
- Build command: `npm install`
- Start command: `npm start`

## Includes
- Dynamic article grid
- Search + category filter
- Blog post routes
- About, Contact, Rules, Terms, Privacy pages
- Contact form API
- Brand logo in SVG
- Mobile-first SaaS-style UI

## Folder structure
- `server.js` — backend + routes
- `public/index.html` — frontend app
- `public/assets/logo.svg` — brand logo
- `public/assets/favicon.svg` — favicon-style icon
